15:Tbe7,<h2>Webpack 是什麼？</h2>
<p>Webpack 是一個現代 JavaScript 應用程式的靜態模組打包工具。當 Webpack 處理應用程式時，它會在內部建立一個依賴圖，此依賴圖映射專案所需的每個模組，然後生成一個或多個 bundle。</p>
<h2>為什麼要使用 Webpack？</h2>
<p>在現代前端開發中，我們面臨以下挑戰：</p>
<ul>
<li>管理大量 JavaScript 文件和依賴關係</li>
<li>處理不同格式的資源（CSS、圖片、字體等）</li>
<li>提高載入效率和性能</li>
<li>開發與生產環境的差異配置</li>
</ul>
<p>Webpack 解決了這些問題，它讓我們能夠模組化開發，並高效地打包所有資源。</p>
<h2>安裝與基本配置</h2>
<p>首先，讓我們建立一個新專案並安裝 Webpack：</p>
<pre><code class="language-bash">mkdir my-webpack-project
cd my-webpack-project
npm init -y
npm install webpack webpack-cli --save-dev
</code></pre>
<p>安裝完成後，我們可以在專案根目錄建立一個 <code>webpack.config.js</code> 文件：</p>
<pre><code class="language-javascript">const path = require("path");

module.exports = {
  entry: "./src/index.js",
  output: {
    filename: "main.js",
    path: path.resolve(__dirname, "dist"),
  },
};
</code></pre>
<p>這個配置檔告訴 Webpack：</p>
<ol>
<li>打包的入口點是 <code>./src/index.js</code></li>
<li>輸出的檔案應該放在 <code>./dist</code> 目錄下，命名為 <code>main.js</code></li>
</ol>
<h2>創建簡單的應用程式</h2>
<p>讓我們建立一個簡單的 <code>src/index.js</code> 檔案：</p>
<pre><code class="language-javascript">function component() {
  const element = document.createElement("div");
  element.innerHTML = "你好，Webpack";
  return element;
}

document.body.appendChild(component());
</code></pre>
<p>接著，我們建立一個 <code>dist/index.html</code> 檔案：</p>
<pre><code class="language-html">&#x3C;!DOCTYPE html>
&#x3C;html>
  &#x3C;head>
    &#x3C;title>Webpack 入門&#x3C;/title>
  &#x3C;/head>
  &#x3C;body>
    &#x3C;script src="main.js">&#x3C;/script>
  &#x3C;/body>
&#x3C;/html>
</code></pre>
<h2>運行 Webpack</h2>
<p>在 <code>package.json</code> 中添加 script 命令：</p>
<pre><code class="language-json">"scripts": {
  "build": "webpack"
}
</code></pre>
<p>執行：</p>
<pre><code class="language-bash">npm run build
</code></pre>
<p>現在，打開 <code>dist/index.html</code>，你應該能看到 "你好，Webpack" 顯示在頁面上。</p>
